# HTML

- 웹 사이트에서 각 요소가 무엇을 뜻하는 지 알려주는 언어
- tag를 이용해서 작성

# Tags

- '<이름 속성="값">내용</이름>'
- example > <a></a>, <title></title>

# meta

- Extra information for browser
  1. charset
  2. name="author"
  3. name="description"

# Semetic & Non-Sementic

- Semetic

* 요소의 의미를 담고 있는 태그
* <section>, <article>

- Non-Sementic

* 의미없는 태그
* <div>, <span>

# Tag's name

- 여러개의 동일한 태그나 태그가 표현하는 의미에 따라 이름을 정할 수 있다.
- id : 하나의 태그당 고유한 이름, class : 동일한 속성을 가지는 태그들에게 줄 수 있는 이름
